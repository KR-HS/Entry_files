#요청질문
당신은 현직 IT기업 채용담당자이자 백엔드 개발자 채용 경험이 풍부한 면접관입니다.  백엔드 개발자 입사지원서의 자기소개서 항목을 작성해주세요. 

[기본정보]
- 학교 : 고려대학교(세종)
- 전공 : 빅데이터전공
- 학점 : 3.730/4.5
- 자격증 : 정보처리기사, SQLD, ADSP, 운전면허증(1급), 마이크로소프트 (엑셀, ppt, 워드)
- 사회경험 : 아이엠씨큐브(13개월)(마케팅사업부)<국무총리실, 기획재정부 등 SNS컨텐츠 관리,담당>, 
- 직무경험 : 개발 직무 경험은 없지만 관련 교육 수료( Java기반 Spring, AWS, React 등을 활용한 프로젝트 수행 및 전반적인 최신 기술 수강)
- 기술스택 : Java, Spring Boot, Mybatis/JPA, REST API , AWS <EC2,RDS, S3>, Git, SQL<Mysql, Postgresql, Oracle>, WebSocket, WebRTC, Jenkins, React, HTMl, CSS, JS 
- 주요프로젝트 : 
1) 실시간 경매플랫폼 : 실시간 채팅, WebSocket과 WebRTC를이용한 1:N구조의 화상기반 경매 시스템 구축, 중계서버를 두어 경매 관련 데이터 관리
2) 실시간 통학버스 위치관리 시스템 : WebSocket을 이용해 자녀들의 실시간 통학버스의 위치를 공유하고 부모 앱에서는 실시간 위치를 확인할수있고, 자녀들의 하차여부, 유치원 정보 및 위치를 지도상에서 확인가능하며 기사앱에서는 통학버스의 실시간 위치정보를 전송하고 유치원생들의 하차여부 및 통학버스의 운행여부를 관리.
- 강점 : 끈기와 성실성, 문제해결 능력, 업무에 대한 빠른 이해력, 경험을 통한 성장, 코드를 빠르게 파악하는 능력 
- 성격 장점 : 꼼꼼함, 성실함, 협업능력, 상대방 입장에서 생각함.
